## Architecture

MVVM with CommunityToolkit.Mvvm. Manual composition in App.xaml.cs.

Startup: splash → scan → main window (or CLI mode if args present).
Dark titlebar + app icon set via class handler on all windows.

9 windows: MainWindow, RegisteredFilesWindow, OrphanedFilesWindow,
SettingsWindow, AboutWindow, SplashWindow, ConfirmDeleteWindow,
ConfirmMoveWindow, ShortcutOfferWindow.

CLI: /d (delete), /m (move to saved location), /m PATH (move to path).
AttachConsole for output. Exit codes 0 (success) / 1 (error).

## How to run

**Development (elevated terminal):**
```
dotnet run --project src/SimpleWindowsInstallerCleaner
```

**Production (explorer):**
```
src/SimpleWindowsInstallerCleaner/bin/Debug/net8.0-windows/InstallerClean.exe
```
Running the exe triggers UAC elevation automatically (app manifest).

**CLI (elevated terminal):**
```
InstallerClean.exe /d              Delete orphaned files (Recycle Bin)
InstallerClean.exe /m              Move to saved default location
InstallerClean.exe /m D:\backup    Move to specified path
InstallerClean.exe --help          Show usage
```
